# PRD

## 문제정의
한 번의 여행을 위해 평균 11개 이상의 채널을 탐색하며 20시간 이상을 소요하는 등 시간이 오래 걸림
광고성 글과 진짜 후기를 구분하기 어려움

## 목표 사용자
시간 효율을 중시하는 2030 직장인 얼리어답터

## 핵심 가치 제안
파편화된 여행 정보와 인플루언서의 꿀팁을 AX(AI 전환) 기술로 통합하여, 취향과 상황에 맞춘 '실패 없는' 초개인화 여행 및 세미패키지 일정을 3초 만에 자동 생성한다.  

## 핵심 시나리오
사용자가 로그인 후 목적지, 일정, 여행 취향/컨셉(예: 미식, 나홀로 등)을 입력하면 , AI가 사용자 데이터를 분석하여 동선이 포함된 지도 기반의 개인화 일정표를 제공하고 , 인플루언서 피드에서 트렌디한 코스를 원클릭으로 내 일정에 연동한다.  

## MVP 기능 범위
1. 여행 조건 입력 (목적지, 기간, 동반자/취향 컨셉 설정) 
2. AI 기반 초개인화 맞춤 여행 코스 추천 및 지도 UI 동선 시각화 3. 인플루언서 추천 코스 원클릭 내 일정표 연동 및 드래그 앤 드롭 편집 
4. 맞춤형 세미패키지 추천 및 중개 예약 연동   

## 제외할 기능
항공권, 숙박, 투어 상품의 직접 결제 및 API 연동

## 주요 데이터
users (고객 취향 및 스타일 데이터) 
destinations (여행지 정보 및 인플루언서 독점 콘텐츠) itineraries (생성된 여행 일정 및 동선 데이터)
partners (OTA/현지 여행사 상품 및 수수료 데이터)  

## KPI
- AI Recommendation Match Rate: 유저가 최종 선택(저장)한 추천 코스 수 / AI가 자동 생성한 총 추천 코스 수 / 목표 80%
- KPI (지표명)계산식목표AI Recommendation Match Rate(AI 코스 매칭 정확도)유저가 최종 선택(저장)한 추천 코스 수 / AI가 자동 생성한 총 추천 코스 수   80%Influencer Course Sync Rate: 인플루언서 피드에서 '내 일정으로 가져오기'를 클릭하여 연동한 유저 수 / 전체 플랫폼 방문 유저 수 / 목표 50%
- Paid Subscription Conversion Rate: 월 구독료(9,900원) 결제 완료자 수 / 플랫폼 유입 방문자 수 / 목표 5%

## 리스크
1. 초기 브랜드 인지도 부족 및 인플루언서/데이터 확보를 위한 높은 초기 비용 발생 2. 크롤링 데이터 중 과도한 광고성 게시물로 인한 AI 추천 신뢰도 저하 리스크 3. 특수 목적지나 독특한 취향 조합 입력 시 100% 일치하는 데이터 부족으로 인한 생성 한계

## 향후 확장 방향
1. 외부 인플루언서 SNS(인스타그램, 유튜브) 및 숏폼 콘텐츠 API 연계 마케팅 자동화 2. AI 챗봇 기반의 24시간 실시간 여행 컨시어지 기능 도입 3. 단순 일정을 넘어 항공/숙박 및 현지 세미패키지 원스톱 예약·결제 중개 수수료 모델 활성화
