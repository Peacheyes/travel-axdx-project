
# Experiment Plan

## 1. 실험 가설
AX 기반 맞춤 코스 추천 결과를 3가지 테마로 제공하면, 사용자의 여행 일정 선택 및 저장 행동이 증가한다.


## 2. 실험군과 대조군
- Control: 사용자가 목적지와 일정만 입력하면 일반적인 추천 여행지 목록만 제공한다.

- Variant: 사용자가 목적지, 일정, 동반자, 여행 컨셉을 입력하면 5초 이내에 3가지 테마별 맞춤 일정표와 지도 동선을 제공한다.


## 3. 주요 지표
추천 코스 저장률, 일정 생성 완료율, 인플루언서 코스 추가율, 추천 결과 이탈률


## 4. 성공 기준
Variant가 Control 대비 추천 코스 저장률이 10% 이상 높고, 추천 결과 화면 이탈률이 감소하면 성공으로 판단한다.

## 5. 데이터 수집
코스 생성 버튼 클릭, 추천 결과 출력, 추천 코스 저장, 인플루언서 코스 일정 추가 이벤트를 수집한다. 각 이벤트에는 목적지, 여행 일정, 동반자 유형, 여행 컨셉, 선택한 코스 ID, 저장 여부를 함께 기록한다.
