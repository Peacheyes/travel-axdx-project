
# User Journey Map

| 단계 | 사용자 행동 | 감정 | Pain Point | 서비스 기회 |
|---|---|---|---|---|
| 여행 욕구 발생 | 강의계획서 다운로드NS·유튜브에서 여행 콘텐츠 탐색 | 기대감 | 정보가 너무 많고 광고성 콘텐츠 혼재 | 인플루언서 기반 신뢰도 높은 콘텐츠 추천 |
| 여행 계획 시작   | 목적지·일정 검색 | 막막함 | 여러 앱/블로그를 동시에 비교해야 함 | 플랫폼 내 통합 검색 제공 |
| 여행 스타일 입력 | 여행 컨셉, 예산, 동행자 정보 입력 | 귀찮음  | 취향 반영이 어려움 | AX가 사용자 취향 및 동행자 특성 분석 |
| 일정 생성 요청 | 일정 생성 버튼 클릭 | 기대감 | 직접 동선 짜는 데 시간 오래 걸림 | 5초 내 맞춤형 코스 자동 생성 |
| 추천 일정 비교 | AI 추천 코스 3가지 비교  | 만족감 | 어떤 일정이 나에게 맞는지 판단 어려움 | 테마별 일정 + 예상 비용 + 이동 동선 제공 |
| 인플루언서 코스 확인  | 인기 여행 크리에이터 코스 열람 | 신뢰 | 실제 검증된 코스 찾기 어려움 | 원클릭 일정 복사 기능 제공  |
| 일정 수정 및 저장 | 일부 장소 교체·시간 조정  | 편안함 | 수정 시 전체 동선 다시 짜야 함 | AI가 실시간 동선 자동 재조정 |
| 예약 진행 | 숙소·투어·맛집 예약   | 안정감 | 예약 사이트 이동 반복 |  플랫폼 내 예약 연동 |
| 여행 중 사용 | 실시간 일정 확인 및 추천 받기 | 편리함 | 현지 변수 대응 어려움 | 날씨·혼잡도 기반 실시간 추천 |
| 여행 종료 후 | 후기 작성 및 코스 공유 | 뿌듯함 | 후기 작성 번거로움   | 자동 여행 기록·콘텐츠 생성 |
