
## Project Goal [Guides]
2주 안에 AX 기반 초개인화 여행 큐레이션 플랫폼 MVP를 구현한다. 사용자가 목적지, 일정, 동반자, 여행 컨셉을 입력하면 5초 이내에 3가지 테마별 맞춤 여행 일정표와 지도 기반 동선을 확인할 수 있도록 한다. 핵심 기능은 맞춤 여행 코스 생성, 지도 기반 동선 확인, 인플루언서 코스 연동, 필수 입력값 검증이다.

## Guides
팀 프로젝트 기획서, 문제 정의 및 가치 제안, 사용자 시나리오, Acceptance Criteria 문서를 먼저 확인한다. 서비스의 핵심 타겟은 제한된 시간 안에 맞춤형 여행을 원하는 2030 직장인 세대이다. 개발 시 우선순위는 1) 필수 입력값 검증, 2) 5초 이내 추천 결과 출력, 3) 3가지 테마별 일정표 생성, 4) 지도 기반 동선 표시, 5) 인플루언서 코스 연동 순서로 진행한다. 화면과 기능은 발표 시연이 가능하도록 직관적이고 단순하게 구성한다.

## Tools

## Sensors
브라우저 화면 확인, 필수 입력값 검증 확인, 5초 이내 추천 결과 출력 확인, 3가지 테마별 일정표 출력 확인, 지도 동선 표시 확인, 인플루언서 코스 추가 동작 확인, Chrome 개발자도구 콘솔 오류 확인


## Permissions
핵심 요구사항 임의 변경 금지, AC 삭제 금지, 필수 입력값 검증 기능 삭제 금지, 5초 이내 추천 결과 출력 기준 변경 금지, 외부 API 키 하드코딩 금지, 개인정보 하드코딩 금지, 발표 시연에 필요 없는 복잡한 기능 임의 추가 금지


## Feedback Loops
AC 미충족 확인 → 오류 원인 기록 → 코드 또는 화면 수정 → 브라우저 재확인 → 동일 사용자 시나리오 재실행 → 문제가 반복되면 기능 범위를 단순화하여 MVP 시연 가능 상태로 복구
